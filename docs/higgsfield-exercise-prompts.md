# Prompts vidéo par exercice — Higgsfield / text-to-video

Généré automatiquement (62 exercices). Colle chaque prompt dans Higgsfield (sur claude.ai ou l'app), génère un clip **~5 s, fond blanc studio, boucle**, puis **télécharge le fichier en le nommant `<slug>.mp4`**.

Ensuite : mets tous les `.mp4` dans un dossier et lance :

```bash
node scripts/import-exercise-videos.mjs "<dossier>" --commit && git push
```

> Réglages Higgsfield conseillés : format vertical 9:16 ou carré 1:1, 5 s, fond blanc, plan large (full body). Reste cohérent (même style d'athlète/éclairage) sur tous les clips.
