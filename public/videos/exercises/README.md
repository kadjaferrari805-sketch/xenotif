# Vidéos de démonstration des exercices

Dépose ici une vidéo `.mp4` (fond blanc studio conseillé) nommée par le **slug**
de l'exercice — elle sera jouée automatiquement sur la fiche `/exercices/<slug>`.

- Vidéo : `public/videos/exercises/<slug>.mp4`
- Poster (optionnel) : `public/videos/exercises/<slug>.jpg`
- Si aucune vidéo → repli automatique sur le GIF d'animation.

Slugs : voir `exerciceSlugs()` (ex. `squat`, `pompes`, `developpe-couche`,
`developpe-militaire`, `souleve-de-terre`, `curl-biceps`, `tractions`…).
Conseil : muet, ~5-10 s en boucle, ≤ 2 Mo (compresser en H.264 faststart).
